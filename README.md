# 🎵 Music Player Web App

A responsive and interactive **Music Player** built using HTML, CSS, and JavaScript.
It enables users to play, pause, skip tracks, and navigate through a playlist with a clean and modern UI.

---

## 🚀 Live Demo

👉 https://tanasviattri5075-tech.github.io/Music-Player-/

---

## ✨ Features

* ▶️ Play / Pause music
* ⏭️ Next / Previous track navigation
* 🔁 Repeat & 🔀 Shuffle modes
* 📊 Progress bar with seek functionality
* 🖼️ Dynamic song details (image, name, artist)
* ⏱️ Real-time duration & playback tracking

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript (ES6)** – Functionality & Logic
